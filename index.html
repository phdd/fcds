Groovy?

* JVM based superset of Java * dynamic & optionally typed * features are e.g. + Closures + Categories + Builders + [many more](http://docs.groovy-lang.org/latest/html/documentation/)

Java 


List<Integer> numbers = new ArrayList<Integer>();
List<Integer> lessThanThree = new ArrayList<Integer>();

numbers.add(1); numbers.add(2); numbers.add(3); numbers.add(4);

for (String s: numbers) {
    if (s.length() < 3) {
        lessThanThree.add(s);
    }
}

Groovy


def numbers = 1..6
def lessThanThree = numbers.findAll { it.size < 3 }
